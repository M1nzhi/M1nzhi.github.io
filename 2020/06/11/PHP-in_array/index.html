PHP安全(in_array)

in_array

(PHP 4, PHP 5, PHP 7)

in_array — 检查数组中是否存在某个值

说明

大海捞针，在大海（haystack）中搜索针（ needle），如果没有设置 strict 则使用宽松的比较。

参数

$needle: 待搜索的值。

$haystack: 待搜索的数组。

$strict : 如果第三个参数 strict 的值为 TRUE 则 in_array() 函数还会检查 needle 的类型是否和 haystack 中的相同。
